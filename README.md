# vulkan-renderer
A simple project to take up my free time.

# Build Instructions
This project uses [xmake](https://xmake.io/#/) as the build system.

Clone the repository
```
$ git clone https://github.com/alinivar/vulkan-renderer
$ cd vulkan-renderer
```

Build
```
$ xmake
```

Run
```
$ xmake run
```
